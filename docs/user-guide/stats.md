# Statistical Inference

!!! info "Coming soon"
    This page is currently under construction. The `stats` module provides shared
    statistical inference utilities for ConfUSIus analyses.

    **Multiple-comparison correction:**

    - [`adjust_pvalues`][confusius.stats.adjust_pvalues]: Adjust a p-value map for multiple comparisons.

    **Statistical-map thresholding:**

    - [`apply_statistical_threshold`][confusius.stats.apply_statistical_threshold]: Threshold a z-scaled statistical map with multiple-comparison correction and an optional cluster-extent threshold.

    Please refer to the [API Reference](../api/stats.md) for more information.
